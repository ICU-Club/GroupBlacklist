# GroupBlacklist 群组黑名单

- 作者: 星梦
- 出处: `https://github.com/ICU-Club`
- 禁止指定用户组进入服务器，在加载世界前拦截并显示提示
- 支持豁免名单，在线玩家定时检测踢出


## 指令

| 语法 | 权限 | 说明 |
|-----|:----:|------|
| /reload | tshock.cfg.reload | 重载黑名单配置 |

## 配置
> 配置文件路径：tshock/GroupBlacklist.json

```json5

{
  "插件设置": {
    "拒绝加入提示信息": "你的用户组被禁止进入此服务器",
    "踢出提示信息": "你所属的用户组已被列入黑名单",
    "检测间隔(秒)": 10,
    "是否踢出在线黑名单玩家": true,
    "是否记录日志": true
  },
  "黑名单组列表": [
    "superadmin",
    "挂哥"
  ],
  "豁免玩家列表": [
    "AdminName"
  ]
}

```
